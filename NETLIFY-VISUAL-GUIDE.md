### الخطوة 3️⃣: اسحب الملف

```

│          │                │                 │
│  📄      │   ═══════════> │   📁 Drop zone  │

│ .html    │                │   or browse     │

```


```
✅ الحل:
• تأكد من الإنترنت
• افتح Console (F12) وشاهد الأخطاء
• ارفع الملف من جديد
```

```
✅ الحل:
• لا تستخدم وضع التصفح الخاص
```
